# Analysis Walkthrough

What you should learn here:
- Response time vs throughput relationship
- Why averages can hide tail latency
- How to spot errors and correlate with transaction spikes

Minimum graphs to review:
- Average Transaction Response Time
- Transaction Summary
- Running Vusers
- Hits per Second (or Throughput)

Practice:
- Change workload weights
- Rerun
- Compare transaction mix and response time deltas
